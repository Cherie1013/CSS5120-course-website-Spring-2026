# Evaluating LLMs on Chinese High-Context Social Nuances 

This repository contains the automated data collection pipeline, prompt injection scripts (Zero/Weak/Strong), and evaluation materials for measuring the cultural intelligence of Large Language Models (GPT-4o, DeepSeek-V3, Qwen-Max, ERNIE-4.0) in Chinese high-context social scenarios.

##  Dependencies and Environments Required

- **Python Version:** Python 3.8 or higher.
- **Development Environment:** Cursor or Windsurf (recommended for optimal code navigation).
- **Required Libraries:** 
  You can install the necessary Python packages using pip:
  ```bash
  pip install pandas requests openai dashscope numpy matplotlib seaborn
